## Кратко о проекте

Гибридная интеллектуальная система управления распределительными электрическими сетями (ИСУ РЭС), сочетающая:
- **CatBoost** — прогнозирование нагрузки и аварийности (градиентный бустинг)
- **Экспертная система** — продукционные правила (ЕСЛИ–ТО) для принятия решений
- **Трёхуровневая архитектура** — полевой / сетевой / корпоративный уровень
- **Режим «советчика»** — рекомендации диспетчеру через SCADA (без автоматического управления)

### Цель

Проектирование архитектуры, разработка алгоритмического и программного обеспечения ИСУ РЭС, а также формирование методики тестирования и дорожной карты пилотного внедрения на базе Ишимбайских распределительных сетей.

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| ML-модели | CatBoost, градиентный бустинг |
| Базы данных | PostgreSQL, TimescaleDB / InfluxDB |
| Инфраструктура | Docker, Kubernetes |
| Протоколы | MQTT (телеметрия), REST/gRPC (управление), OPC UA (SCADA) |
| Форматы данных | JSON, Protobuf |
| Безопасность | ФЗ-187 (КИИ), многофакторная аутентификация, TLS |
| Стандарты | ГОСТ 32144-2021, ГОСТ Р МЭК 61968-1-2019 |
