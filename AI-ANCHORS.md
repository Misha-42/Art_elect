# AI-ANCHORS.md · Art_elect — Ключевые якоря проекта

## Якорь 1: Тема и объект

| Поле | Значение |
|------|----------|
| Тема НИР | Интеллектуальная система управления Ишимбайскими РЭС |
| Объект | Распределительные электрические сети г. Ишимбай (ООО «Башкирэнерго») |
| Тип системы | Гибридная: ML (CatBoost) + Экспертная система (продукционные правила) |
| Режим работы | «Советчик» (рекомендации диспетчеру, не автоматическое управление) |

## Якорь 2: Трёхуровневая архитектура

| Уровень | Состав | Функция |
|---------|--------|---------|
| Полевой | RTU, устройства РЗА, интеллектуальные датчики | Сбор телеметрии (1с–1мин) |
| Сетевой | Промышленные шлюзы, Modbus→OPC UA, буферизация 72ч | Агрегация, фильтрация, маршрутизация |
| Корпоративный | PostgreSQL/TimescaleDB, Docker/K8s, CatBoost, Expert System | Хранение, ML, принятие решений |

## Якорь 3: Функциональные модули

1. **Мониторинг** — MQTT-шина, валидация, буферизация при потере связи
2. **Диагностика** — правила + статистика → норма/предупреждение/авария/пост-авария
3. **Прогнозирование** — CatBoost: нагрузка (1–168ч), вероятность отказов (доверительные интервалы)
4. **Управление** — база знаний (ЕСЛИ–ТО), гибридный вывод, верификация по ПТЭЭП/ГОСТ

## Якорь 4: Стандарты и регуляторика

- **ГОСТ 32144-2021** — качество электроэнергии
- **ГОСТ Р МЭК 61968-1-2019** — интеграция приложений управления РЭС
- **ФЗ-187** — безопасность объектов КИИ
- **Приказ Минэнерго №1044** — АИИС КУЭ
- **ПТЭЭП** — правила технической эксплуатации электроустановок
