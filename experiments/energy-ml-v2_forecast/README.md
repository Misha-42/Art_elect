# energy-ml-v2_forecast — Прогнозирование нагрузки (CatBoost)

> Раздел 2.3 НИР-2: модуль прогнозирования на базе градиентного бустинга.

## Цель
Прогнозирование активной мощности P_feeder_1 на горизонты:
- **1 час** (10 шагов) — оперативное управление
- **4 часа** (40 шагов) — тактическое планирование
- **24 часа** (240 шагов) — суточное планирование
- **168 часов** (1680 шагов) — недельное планирование

## Метрики
- **MAE** — средняя абсолютная ошибка (кВт)
- **RMSE** — корень из среднеквадратичной ошибки (кВт)
- **R²** — коэффициент детерминации
- **MAPE** — средняя абсолютная процентная ошибка (%)

## Модель
**CatBoostRegressor** с подбором гиперпараметров (Random Search):
- глубина деревьев: 4–8
- скорость обучения: 0.01–0.3
- L2-регуляризация: 1–10
- количество итераций: до 1000 с early stopping
