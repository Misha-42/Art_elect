# energy-ml-v5_control — Замкнутое управление (CatBoost + ЭС)

> Раздел 2.5 НИР-2: алгоритмы автоматизации восстановления питания и оптимизации режимов сети.

## Архитектура

```
[Телеметрия] → [CatBoost (прогноз режима)] → [Экспертная система (проверка)]
                                                    ↓
[Исполнение] ← [Верификация] ← [Формирование управляющего воздействия]
```

## Компоненты

1. **CatBoost** — классификация 4 режимов (норма/предупредительный/аварийный/пост-аварийный)
2. **Экспертная система** — 24 продукционных правила (приоритет безопасности)
3. **Гибридный решатель** — приоритет critical-правил над ML
4. **Журнал аудита** — фиксация всех решений
5. **Метрики** — BA, MCC, F1, точность по классам

## Запуск

```bash
python code/run_closed_loop.py
```
