## Формат телеметрии (предлагаемый)

| Колонка | Тип | Описание |
|---------|:---:|----------|
| timestamp | datetime | Метка времени |
| U_10kV_A | float | Напряжение 10 кВ, фидер A |
| U_10kV_B | float | Напряжение 10 кВ, фидер B |
| I_feeder_N | float | Ток фидера N |
| P_feeder_N | float | Активная мощность фидера N |
| Q_feeder_N | float | Реактивная мощность фидера N |
| f_grid | float | Частота сети |
| cosφ | float | Коэффициент мощности |
| alarm_flag | int | Флаг аварийного события (0/1) |
| mode | int | Режим работы: 0=норма, 1=предупреждение, 2=авария |
