# energy-ml-v1_baseline — Классификация режимов сети (baseline)

> Первый ML-эксперимент: базовая классификация режимов распределительной сети.

## Цель
- Сбор и разметка телеметрии Ишимбайских РЭС
- Бинарная/многоклассовая классификация: норма / предупреждение / авария
- Baseline-метрики (логистическая регрессия, случайный лес)

## Структура

```
energy-ml-v1_baseline/
├── datasets/           # Датасеты эксперимента
├── code/               # Скрипты
│   ├── build_dataset.py      # Сбор и разметка
│   ├── run_baseline.py       # Обучение baseline
│   └── run_shap.py           # SHAP-интерпретация
├── models/             # Сохранённые модели
└── results/            # Результаты и метрики
```

## Метрики цели
- Balanced Accuracy ≥ 0.85
- F1 ≥ 0.80
- ROC-AUC ≥ 0.90
