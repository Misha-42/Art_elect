## О проекте

Разработка интеллектуальной системы управления (ИСУ) для распределительных электрических сетей г. Ишимбай. Система сочетает:

- **CatBoost** — прогнозирование нагрузки (1–168 ч) и вероятности отказов
- **Экспертная система** — продукционные правила (ЕСЛИ–ТО) на основе знаний диспетчеров
- **Трёхуровневая архитектура** — полевой → сетевой → корпоративный уровень
- **Режим «советчика»** — рекомендации диспетчеру через SCADA (ФЗ-187)

### Ключевые возможности

- 📊 **Мониторинг** — сбор и валидация телеметрии в реальном времени (MQTT, 1с–1мин)
- 🔍 **Диагностика** — классификация режимов: норма / предупреждение / авария
- 📈 **Прогнозирование** — нагрузка, отказы, доверительные интервалы (CatBoost)
- 🧠 **Экспертная система** — формализованные правила диспетчеров
- 🔒 **Безопасность** — ФЗ-187, многофакторная аутентификация, TLS
