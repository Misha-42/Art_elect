# Соглашения о коммитах (Commit Conventions) · Art_elect

## Префиксы

| Префикс | Тип | Пример |
|---------|-----|--------|
| `[INIT]` | Инициализация | `[INIT] feat: скелет репозитория Art_elect` |
| `[DOC]` | Документация | `[DOC] docs: ROADMAP_10_STEPS — план развития` |
| `[EXP]` | Эксперимент | `[EXP] feat: baseline v1 — классификация режимов` |
| `[FIX]` | Исправление | `[FIX] fix: исправлен leakage в run_baseline` |
| `[OPS]` | Эксплуатация | `[OPS] feat: скрипт мониторинга телеметрии` |
| `[SEC]` | Безопасность | `[SEC] feat: модель угроз ФЗ-187` |
| `[DATA]` | Данные | `[DATA] feat: генератор синтетической телеметрии` |

## Структура

```
<префикс> <тип>: <что сделано>

<результат>:
- Ключевые метрики
- Что решено
- Инсайты
```

## Язык
- Русский (английские слова — с переводом в скобках)
