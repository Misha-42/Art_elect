# DATA_NAVIGATION.md · Art_elect — Источники данных

> Реестр источников данных для ИСУ Ишимбайских РЭС.

---

## SCADA-телеметрия (основной источник)

| Тип данных | Теги | Частота | Источник |
|-----------|------|:-------:|---------|
| Напряжение (U) | U_10кВ, U_0.4кВ | 1с–1мин | RTU, SCADA |
| Ток (I) | I_фидер, I_тр | 1с–1мин | RTU, SCADA |
| Мощность активная (P) | P_фидер, P_тр | 1с–1мин | RTU, SCADA |
| Мощность реактивная (Q) | Q_фидер, Q_тр | 1с–1мин | RTU, SCADA |
| Частота (f) | f_сеть | 1с | RTU, SCADA |
| Коэффициент мощности | cosφ | 1мин | RTU, SCADA |
| Положения выключателей | Q_статус | событие | SCADA |
| Аварийные сигналы | Alarm_* | событие | SCADA, РЗА |

## АИИС КУЭ (коммерческий учёт)

| Тип данных | Период | Частота |
|-----------|:------:|:-------:|
| Почасовое потребление | архив 3+ года | 1 час |
| Суточное потребление | архив 3+ года | 1 сутки |
| Профили нагрузки | архив 3+ года | 30 мин |

## Внешние данные

| Тип | Источник | Формат | Назначение |
|-----|----------|:------:|-----------|
| Метеоданные | Гидрометцентр / API | JSON/CSV | Коррекция прогноза нагрузки |
| Праздники/календарь | Производственный календарь | CSV | Учёт сезонности |
| ГИС | ЦГИС / API | GeoJSON | Топология сети |
| Отказы оборудования | Журнал эксплуатации | XLSX/CSV | Обучение модели отказов |

## Структура данных

```
data/
├── raw/                 # Сырые данные (из SCADA, не изменять)
│   ├── telemetry/       # Телеметрия (CSV/Parquet)
│   ├── aiskue/          # Данные АИИС КУЭ
│   ├── weather/         # Метеоданные
│   └── incidents/       # Журнал аварийных отключений
├── processed/           # Обработанные датасеты (готовы к ML)
└── external/            # Внешние данные (ГИС, календари)
```
