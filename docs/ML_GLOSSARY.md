# ML_GLOSSARY.md · Art_elect — Словарь ML-терминов

> Словарь терминов машинного обучения в контексте управления распределительными сетями.

---

## A–Z

### Градиентный бустинг (Gradient Boosting)
Метод ансамблевого обучения, где последовательно строятся деревья решений, каждое следующее исправляет ошибки предыдущего. CatBoost — реализация Яндекса с автоматической обработкой категориальных признаков.

**В проекте:** основной метод прогнозирования нагрузки и вероятности отказов.

### Инженерия признаков (Feature Engineering)
Процесс создания новых признаков из исходных данных для улучшения качества моделей.

**В проекте:** лаги нагрузки, скользящие окна (среднее, мин, макс за 1ч/8ч/24ч), календарные признаки (час дня, день недели, месяц), метеоданные.

### Кросс-валидация (Cross-Validation)
Метод оценки качества модели, при котором данные делятся на несколько частей, и модель обучается/тестируется на разных комбинациях.

**В проекте:** TimeSeriesSplit (для временных рядов — без shuffle).

### Продукционные правила (Production Rules)
Правила вида «ЕСЛИ условие ТО действие» (IF-THEN), используемые в экспертных системах.

**В проекте:** база знаний экспертной системы для принятия решений по управлению сетью, формализация опыта диспетчеров.

### Режим «советчика» (Advisor Mode)
Режим работы ИСУ, при котором система формирует рекомендации, но окончательное решение принимает диспетчер. Критические операции требуют подтверждения.

**В проекте:** обязательный режим для соблюдения ФЗ-187.

### SHAP (SHapley Additive exPlanations)
Метод интерпретации предсказаний моделей, основанный на теории игр. Показывает вклад каждого признака в предсказание.

**В проекте:** анализ важности телеметрических тегов для классификации режимов.

### ROC-AUC / PR-AUC
- **ROC-AUC** — площадь под ROC-кривой (True Positive Rate vs False Positive Rate)
- **PR-AUC** — площадь под Precision-Recall кривой (важнее для дисбаланса классов)

**В проекте:** оценка качества детекции аномалий и прогноза отказов.
