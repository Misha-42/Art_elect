# TECHNIQUES.md · Art_elect — Журнал освоенных техник

> Техники работы, освоенные в ходе проекта.

---

## 1. Трёхуровневая архитектура ИСУ

**Когда применять:** при проектировании систем управления распределительными сетями
**Суть:** разделение на полевой (сбор), сетевой (агрегация/безопасность) и корпоративный (ML/аналитика) уровни
**Эффект:** масштабируемость, отказоустойчивость, соответствие ФЗ-187

## 2. Гибридная схема ML + экспертная система

**Когда применять:** задачи, где нужна и точность прогнозов, и детерминированность нормативных правил
**Суть:** ML делает прогноз, экспертная система проверяет по правилам и формирует решение
**Эффект:** высокая точность + гарантия соблюдения нормативов

## 3. Data leakage protection для временных рядов

**Когда применять:** любые ML-задачи с временными рядами (телеметрия)
**Суть:** train_test_split(shuffle=False), лаги только из прошлого
**Эффект:** честная оценка качества модели
